Projeto 1: Bem-te-vi

Website interativo que utiliza CSS e JavaScript para criar animações envolventes de um objeto sonoro, representado por um modelo de pássaro. O usuário pode interagir com o pássaro, que reage visualmente e auditivamente, proporcionando uma experiência imersiva de reprodução sonora.

Projeto 2: Conversor Monetário

Este projeto é um conversor de moedas interativo desenvolvido em HTML, CSS e JavaScript. Ele permite que o usuário selecione entre três moedas (Dólar, Real e Euro) e insira o valor a ser convertido. Após escolher as moedas de origem e destino, o usuário pode clicar em um botão para calcular o valor convertido. A interface possui um design limpo com campos fáceis de usar e um resultado exibido de forma imediata.
